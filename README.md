# Сыпачев Иван КМБ-2 Лабараторная №5
# Задание 1
## Задача 10
### Текст задачи
Дерево содержит целые числа. К каждому числу приписать заданную цифру в
конец.
### Алгоритм решения
Программа запрашивает у пользователя глубину дерева. Ввод проверяется с помощью Int32.TryParse:
Если введено целое число, оно передается в функцию генерации.
В случае ввода текста или некорректных символов выводится ошибка «Введена не цифра».
Рекурсивная функция generateRandomTree:
Базовый случай: если глубина меньше или равна 0, функция возвращает Empty (пустой узел).
Рекурсивный шаг: создается узел Node со случайным числом от 1 до 99, а его левая и правая ветви заполняются результатами повторного вызова этой же функции с уменьшенным значением глубины.
Далее выводится исходное дерево. Для этого используется рекурсивная функция printTree, которая обходит дерево справа налево и печатает значения с отступами для визуализации иерархии.
Программа просит ввести цифру для добавления в конец каждого числа. Ввод проверяется через Int32.TryParse:
Если введена цифра от 0 до 9, запускается трансформация дерева.
В противном случае выводится ошибка.
Функция Tree.map:
Базовый случай: если встречен пустой узел Empty, возвращается Empty.
Рекурсивный шаг: к значению текущего узла применяется лямбда-выражение (v * 10 + digit), которое математически приписывает цифру в конец числа. Затем функция рекурсивно вызывает map для левого и правого поддеревьев, формируя новое дерево.
В конце программа повторно вызывает функцию printTree для отображения трансформированного дерева с измененными значениями.

# Задание 2
## Задача 10
### Текст задачи
Сколько элементов дерева не содержит заданную цифру?
### Алгоритм решения
Программа запрашивает у пользователя глубину дерева. Ввод проверяется с помощью Int32.TryParse:
Если введено целое число, оно передается в функцию генерации дерева.
В случае некорректного ввода выводится сообщение «Введена не цифра».
Рекурсивная функция generateRandomTree:
Базовый случай: если глубина стала меньше или равна 0, возвращается Empty.
Рекурсивный шаг: создается узел Node со случайным числом от 1 до 99, а его левая и правая ветви заполняются результатами повторного рекурсивного вызова с уменьшенной глубиной.
Далее программа выводит структуру дерева с помощью рекурсивной функции printTree, которая использует отступы для отображения иерархии (обход справа налево).
Затем программа запрашивает цифру, которую нужно искать в числах. Ввод проверяется через Int32.TryParse:
Если введена цифра от 0 до 9, начинается процесс подсчета.
В противном случае выводится ошибка.
Функция containsDigit:
Преобразует текущее число и искомую цифру в строки и проверяет вхождение одной строки в другую с помощью метода .Contains.
Рекурсивная функция Tree.fold (свёртка):
Базовый случай: для пустого узла Empty возвращает текущее накопленное значение (аккумулятор).
Рекурсивный шаг: сначала функция применяет переданное лямбда-выражение к текущему значению узла, а затем последовательно передает обновленный аккумулятор в левое и правое поддеревья.
Процесс подсчета в main:
Используется Tree.fold с начальным значением 0.
Внутри лямбда-функции для каждого числа вызывается containsDigit.
Если цифра не найдена (if not ...), аккумулятор увеличивается на 1, в противном случае значение сохраняется.
В конце программа выводит итоговое количество элементов, в которых заданная цифра отсутствует.
